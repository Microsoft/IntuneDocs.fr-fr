# L’appareil n’est pas conforme à la version maximale du système d’exploitation

Votre appareil iOS n’est pas conforme à la stratégie de votre administrateur informatique pour la version maximale du système d’exploitation. Contactez votre administrateur informatique. Pour obtenir ses coordonnées, consultez le [site web du Portail d’entreprise](http://portal.manage.microsoft.com).
