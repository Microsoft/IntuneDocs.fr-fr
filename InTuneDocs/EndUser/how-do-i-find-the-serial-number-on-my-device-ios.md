# Comment trouver le numéro de série sur mon appareil ?

Sur les appareils exécutant iOS versions 7.1 et ultérieures, appuyez sur l’icône **Paramètres**, sur **Général** > **À propos de**, puis faites défiler vers le bas jusqu’à **Numéro de série**.
